# 智能電商商品圖對齊優化服務 (Smart E-commerce Image Alignment)

本工具旨在自動化處理電商商品圖片，確保所有圖片皆符合統一的視覺標準與邊界規範。

## ✨ 功能特色

-   **自動對齊**：將商品底部精準對齊至 **Y=900**。
-   **邊界限制**：確保商品嚴格位於左側 (X=70)、右側 (X=930) 與頂部 (Y=80) 的安全範圍內。
-   **智慧縮放**：自動計算最佳縮放比例，在遵守所有邊界限制的前提下，最大化商品展示尺寸。
-   **標準輸出**：生成 **1000x1000 px** 的白底 JPG 圖片，統一格式。
-   **格式支援**：支援 **JPG** 與 **WebP** 格式輸入，輸出統一為 JPG。
-   **雙重模式**：提供「網頁視覺化介面」與「資料夾批次處理」兩種使用方式。

## 🛠️ 系統需求

-   Python 3.x
-   OpenCV (`opencv-python-headless`)
-   NumPy
-   Streamlit
-   Pillow

## 📥 安裝說明

1.  安裝所需套件：
    ```bash
    pip install -r requirements.txt
    ```

## 🚀 使用說明

### 方式一：網頁介面 (推薦)
適合需要即時預覽或處理少量圖片的使用者。

1.  啟動網頁應用程式：
    ```bash
    streamlit run app.py
    ```
2.  瀏覽器將自動開啟 (預設網址為 `http://localhost:8501`)。
3.  將圖片拖曳至上傳區，即可即時預覽處理前後的對比。
4.  點擊下載按鈕，取得包含所有處理後圖片的 ZIP 壓縮檔。

### 方式二：批次處理腳本
適合需要一次處理大量圖片的使用者。

1.  將您的商品圖片 (JPG 或 WebP 格式，建議白底) 放入專案目錄下的 `input` 資料夾。
2.  執行主程式：
    ```bash
    python main.py
    ```
3.  處理完成的圖片將自動儲存於 `output` 資料夾中。

## 📏 規範細節 (Constraints Details)

-   **輸出尺寸**：1000 x 1000 px
-   **底部邊界 (Bottom)**：Y = 900 px (**最高優先級**)
-   **頂部邊界 (Top)**：Y >= 80 px
-   **左右邊界 (Side)**：70 px <= X <= 930 px
-   **檔案大小限制**：輸出 < 400KB
